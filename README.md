    - **城市产业（城市经营一期）**：除对马岛外每座城市各有一处可置办的**私人产业**，类型按城市风味固定五选一——🍶酒馆（1200 金置办，日进 35 金）、🐎马场（1600 金，日进 48 金）、⛏️矿山（800 金，日进 18 金，且每满 5 天额外积攒 1 份本城铁匠铺专精材料，材料待收上限 3 份）、🌾农庄（800 金，日进 25 金）、🚢商队（2000 金，日进 60 金，但收取时按行情浮动六成～一倍四）；置办耗 1 行动力，收益按天自动累积但**须本人到城收取**（收取不耗行动力），最多积攒 **15 天**便停止累计——常回来巡视才是正道。可为每处产业**委任掌柜**：人选须是**友谊满上限（300）且不在团队中**的已现身武将（敌我阵营皆可），收益加成 =（智力+政治）÷4、封顶 +50%，掌柜走马上任后常驻该城打理产业（不再随宿营云游，也不随败军撤离；被招募入队则自动卸任）；**掌柜还会逐日代收金币直接入你的钱囊**——无须亲至、不受 15 天积攒上限（商队的行情浮动照常生效），面板上可查累计代收额（金币与材料分列）；矿山材料同样由掌柜逐日一并代收直接入库，不再受待收上限约束。**亲自收取时有小概率触发经营事件**：🗡️ 强人劫掠（12%，农庄安稳无此虞）——应战（真实经典单挑）获胜追回**双倍**进账、落败只保住一半，或选择破财消灾保住七成，材料不为强人所图、两种结局均送回宝物库；📈 大市（10%）——本次金币收益翻倍；🎉 贵客临门（10%，酒馆特有）——随机一位己方已现身武将造访畅饮，友谊 +10。掌柜代收的部分静默入账，不触发任何事件。**城池若被敌方阵营攻占，该城产业立即被查封**——停产、不可收取也不可变卖，直至己方在边境战中夺回自动恢复（查封期间的天数颗粒无收，占领前已积攒的待收账款原封保留）；反之，己方新攻占的敌城也随即可以置业。产业随时可**变卖**（非查封状态），回收累计造价（含历次扩建）五成外加全部未收账款。地图左栏角色资料卡下方有「🏪 名下产业」总览行，随时可见持有数量与待收总额。产业数据全存战役层，新开局重置。

## 🚀 运行

任意静态服务器即可（PWA 需 http/https，不能用 `file://` 直接打开）：

```bash
# 方式一：Node http-server
npx http-server -p 8099 -c-1

# 方式二：Python
python -m http.server 8099
```

浏览器打开 `http://localhost:8099`。

## 📱 安装到手机

1. 将整个文件夹部署到任意 HTTPS 静态托管（GitHub Pages / Vercel / Netlify 等）。
2. 手机浏览器打开网址 → 菜单选择「添加到主屏幕 / 安装应用」。
3. 之后可像原生 App 一样离线启动。

## ➕ 丰富武将数据

两种方式：

1. **游戏内**：进入「武将图鉴」→「新增武将」，填写属性即可，自动持久化。
2. **改源码**：编辑 `data/generals.js`，在 `CN_GENERALS` 或 `JP_GENERALS` 数组追加：

```js
{ name: "某某", title: "称号", intro: "简介…",
  ti: 90, wu: 88, tong: 80, zhi: 70, zheng: 60, mei: 75 }
```

`side` 与 `id` 会自动补全。`ti`（体力）同时作为单挑初始 HP。

> 注：游戏内一旦改过数据库会以 `localStorage` 为准；想用源码新增项，可在图鉴页点「恢复默认」刷新。
