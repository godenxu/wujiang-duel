# 中日武将大单挑 ⚔️

三国 VS 战国 —— 中日两国名将捉对厮杀的单挑对决游戏。纯前端 HTML 实现，支持 PWA 安装到手机离线运行。

## ✨ 特性

- **首页 5 大模块（武将扮演为主线）**：
  - **🆕 新游戏**：三步开局向导——① 选角色（自创武将，或选一位史实武将扮演，扮演史实武将时可二选一：**少年模式**初始六维为默认值 60%、最高两项属性定为「本命天赋」历练加点成长 +50%（可突破默认上限）；**巅峰模式**默认原值开局但历练加点成长减半）② 选天下格局（**史实分布**——武将按势力落位，如蜀将现于成都、织田家臣现于尾张；**群雄乱入**——随机洗牌，每局天下大不同）③ 确认开局——确认后清空角色/金币/友谊/队伍/宝物背包等**战役数据**，开启全新的武将人生并直接进入「天下地图」（若已有存档会先提示覆盖确认）。
  - **▶️ 继续游戏**：读取自动存档，按钮上直接显示当前角色名/等级/战绩，无存档时置灰不可点；点击后同样进入天下地图。
  - **🗺️ 天下地图（战役主界面）**：中原二十城 + 战国二十城 + 对马岛海路中转，共 41 个节点的风格化地图，叠加中日陆地剪影背景（风格化手绘轮廓，非精确测绘但取意接近真实海岸线），城池以矩形徽章（姓名直接嵌入其中）标出、SVG 道路连接，点击相邻城池即可移动；支持**缩放拖动**——鼠标/触控拖拽平移、双指捏合或滚轮缩放、右上角四个按钮（＋/－缩放、🎯一键聚焦当前城市放大、🗺️一键复位显示全景），缩放范围 1~3 倍；**行动力**系统——移动、历练、悬赏、设施挑战等各耗 1 点，每日初始 1 点（随名声/武道会首冠/唯一奇珍提升，封顶 6），「🏕️ 宿营」（与历练/集市/铁匠铺/设施同排按钮）推进一天并回满；当前城池面板提供**拜访**（本地已现身武将名录，仅显示完整姓名、每行 3 个；下方是与「🎭 角色详情」同风格的「全部武将」大按钮，打开与武将图鉴同风格的只读表格页面，可按列排序/搜索/三国战国筛选，六维数值实时叠加当前已装备宝物，新增友谊值与所在城池两列，点击任意一行弹出的详情仍展示同阵营武将的友谊进度条，但隐藏拜访/切磋/招募等互动按钮、装备槽仅供查看不可更换）、**历练**、**集市**（每城每日一批本地货摊，价格随城市行情浮动、对马岛黑市恒八折、买断即售罄）、**铁匠铺**（各城专精一类宝物，锻专精类省料省钱）、**悬赏榜**（每城 2~3 条，讨伐令/车轮令/登塔令/双雄令四类穿插，一行 2 条、点击整条即接取（不再有独立接取按钮），完成得金+名声，高级悬赏额外掉唯一奇珍）、**特色设施**（每城一项，天下擂台/车轮武场/百人斩道场/双人比武/军团攻城/远征/武道会预选循环覆盖全部 40 城，胜利额外记名声）内容；顶部标题栏「天下游历」与音乐按钮之间只显示当前游历天数，换算为**天下历**（每年 12 月、每月 30 天）以「X 年 X 月 X 日」呈现且不换行；行动力与拥有金币移至当前城池名称一行右侧对齐显示；名声值（含名声等级、等级在前）移至角色资料卡「🎭 角色详情」按钮上方一行；等级信息在姓名下方、评分左侧，四者互不重复。折叠屏/平板展开（宽度 ≥ 600px）时自动切换为「地图横贯顶部（约占可视高度 30%）+ 下方左（角色资料+本地武将）右（城池事务）两栏」的布局，地图区吸顶固定，滚动城池面板时地图始终可见。武将按选定的天下格局分布在各城，开局每城仅 2~3 位现身，此后每宿营跨越 5 天会揭示一批 3~5 名新武将（「⚡ 天下快报」），不按名声分层，随时可能遇见任何一位高手；此外已现身的武将每次宿营还有 3% 概率自行迁往阵营内相邻城池，天下版图并非一成不变。移动时约 30% 触发赶路奇遇（山贼/奇珍/商队折扣/世外高人/路遇名将/客栈临时行动力），渡海（对马岛沿线）额外 20% 触发风暴（延误或漂流）；对马岛集市常驻黑市八折。全站返回逻辑已同步手机系统/浏览器的返回键——不再只能靠点击左上角箭头，硬件返回会按同一套规则返回上一层（若有弹窗打开则优先关闭弹窗）。
    - **城池归属与边境阵营大战**：城池归属独立于武将分布的固有阵营，可随战事易主；对马岛作为唯一连通两大板块的海路中转站，初始划归战国一方。每逢天下历**月末**（第 30/60/90…天宿营时），归属不同的相邻城池之间会各自爆发一场边境阵营大战——双方各出「已现身武将」中随机等量人马（不限当地驻军），若某场边境正与主角所在阵营相关，可选择率团队一同出征（团队成员必上阵，其余随机抽选），也可选择「各地驻军自行迎战」不亲征；胜方直接夺取败方该城，前线随城池易主逐月推移。
    - **潜入敌境与刺杀**：角色扮演武将只能与己方阵营武将交友——本地武将名录/全部武将详情中，敌方阵营的武将不显示拜访/切磋/招募的交友板块，取而代之的是「🗡️ 刺杀」（经典单挑，每人每游戏日限一次）；刺杀得手，敌将六维随机一项 -1~3（战役内生效、不影响武将图鉴全局数值）；刺杀反被所伤，主角自己六维随机一项 -1~3。宿营过夜时，若当前城池本地武将中恰有敌方阵营成员，也有 15% 概率反过来被其偷袭夜战，胜负结算与主动刺杀完全对称。
  - **⭐ 名声与终局**：九阶名声（无名之辈→略有耳闻→小有名气→声名初显→威震一方→名动一国→威名远播→名满天下→威加四海，数值上限 9999，状态条与角色卡直接显示具体数值），赢切磋、完成悬赏、设施挑战、击败高手、世界杯名次均可积累；每阶都有具体变化——如「小有名气」解锁历练、「威震一方」解锁「天下第一武道会」报名、「名动一国」为每城悬赏榜永久 +1 条、「名满天下」铁匠铺专精锻造再减免；达到「威震一方」方可报名武道会（世界杯 32 强），首次夺冠永久 +1 行动力上限；集齐「威名榜」八大高手（吕布/关羽/张飞/赵云/织田信长/武田信玄/上杉谦信/本多忠胜）的胜绩 + 至少一次武道会夺冠，触发一次性「天下无双」终局庆典，此后照常继续自由游历。**历练**（随机对手自由练级）需名声达「小有名气」才解锁，开局阶段请先靠拜访、**切磋**（武将详情弹窗新增按钮，向本地武将下战书，经典单挑，每人每游戏日限一次、耗 1 行动力，获胜得友谊 +3~5）、悬赏与设施挑战打出声望。
  - **📜 武将图鉴**：全局设定集，展示 400 名武将的**默认**六维与资料，可增删改，但不显示友谊/装备等游戏进度（不受任何一局游戏进度影响）。
  - **🎁 宝物阁**：全局宝物总览，一次性展示全部 100 件宝物模板（不受"发现进度"限制），可编辑名称/简介/属性/各稀有度加成值（限幅 ≤15），也可自建全新宝物——编辑/自建的结果会计入之后每一局新游戏的掉落与商店池，同样不受游戏进度影响。
  - **🕹️ 小游戏**：经典单挑 / 车轮大战 / 百人斩·爬塔 / 2v2 主副将 / 阵营大战 / 组队大战 / 国战·攻城略地 / 武将世界杯共 8 个自由玩法合集，统一使用武将图鉴默认数值，**不消耗、不产出任何存档数据**（金币/经验/友谊/宝物全程不受影响），适合单纯想爽玩一局的场景。
- **存档架构**：分「全局层」（武将图鉴自定义武将、宝物阁模板编辑，永不因新游戏重置）与「战役层」（角色、金币、友谊、队伍、宝物背包，随「新游戏」清空重来）两层，互不干扰。
- **武将数据库**：三国 200 名、战国 200 名（共 400 名真实历史武将），每名武将含 **姓名、称号、简介** 及 **体力 / 武力 / 统帅 / 智力 / 政治 / 魅力** 六维属性。
- **数据库可维护**：图鉴页可 **新增 / 编辑 / 删除** 武将，数据自动存入浏览器 `localStorage`；支持 **导出 / 导入 JSON**、**一键恢复默认**。
- **玩法模块**（既可在「小游戏」中自由畅玩，也会在扮演角色后从「角色扮演」主页发起并计入战役进度）：
  - **经典单挑**：自选双将（或随机双将），轮换出招回合制，亲自指挥每一招；统帅定先手，魅力定暴击。
  - **车轮大战**：一员主将连续挑战由弱到强的敌将，挑战连胜纪录；每阵获胜后无需手动确认，结算浮层短暂展示后自动迎战下一员（也可立即点击提前进入或选择鸣金收兵）。
  - **百人斩·爬塔**：无尽 Roguelike——守将逐层增强、带伤攀塔，每 5 层「天赐机缘」三选一强化，最佳层数存档；普通楼层获胜同样自动攀上一层，无需逐层确认，仅「天赐机缘」三选一仍需手动挑选。
  - **2v2 主副将单挑**：副将六维的 15% 并入主将，主将危急时副将舍身驰援疗伤一次。
  - **阵营大战**：规模可选每方 50 / 100 / 全部 / 随机数量，两军自动捉对厮杀直到一方全灭（可切换详情观战；从角色扮演进入时自选武将必上阵）。
  - **组队大战**：各带兵最多 10 将，带兵攻击 / 计谋（乱阵·伏兵·安抚·驰援）/ 挑唆单挑 / 征兵；兵力条按兵量比例伸展、中间端对齐屏幕中央，受创时从武将名字一侧向中间收缩。
  - **国战·攻城略地**：每局随机生成城池地图（城市数量、双方地盘、驻军皆随机；双方武将总数相同，最多各 200 员、全部从真实武将卡池直选），Gabriel 图随机连通道路；相邻攻伐/调兵、出兵行军动画，攻城战按组队规则每方最多 10 将上阵（超编随机选拔：攻方其余留守出发城，守方其余城内候命、城破时退往相邻友城），上阵败方全军覆没、阵亡不复活，占领全图一统天下。
  - **武将世界杯（32 强）**：随机分组 → 小组循环赛 → 单败淘汰；淘汰赛开打前可填**竞猜预测表**，按命中率计分、猜中冠军翻倍；对阵图折叠屏两种形态自适应（窄屏横向、宽屏上下半区纵向汇聚）。
  - **角色扮演·武将养成**：由「新游戏」开局向导创建（自创或扮演史实武将，见上），历练/车轮/百人斩/2v2/阵营/组队/国战/世界杯全部玩法均可获得经验升级；养成页现为「天下地图」之下的角色详情子页（地图状态条点「🎭 详情」进入，返回固定回地图），内含「🏪 宝物库（仓库·商店·锻造）」入口，即原「宝物库」的战役内实例视图（与首页全局「宝物阁」区分：前者是本局背包与交易，后者是不受进度影响的全局模板编辑）；宝物库也可从地图当前城池的「集市」「铁匠铺」按钮直接进入对应页签。无论从地图还是养成页发起战斗，结算后都会正确返回发起时所在的那一层。
  - **友谊与金币**：主角与 400 名武将并肩/交手皆积累友谊（陌生→相识→好友→挚友→生死之交）；各玩法胜利赚金币；挚友可花金招募入队（生死之交免费）；队友可任 2v2 副将，同阵营队友在组队大战/国战/阵营大战中必上阵；世界杯竞猜得分直接兑换金币；金币便利消费——百人斩机缘 50 金重抽、主角单挑倒地 100 金服「回魂丹」满血续战（每场一次）、国战开局 30 金重掷地图、宝物库鉴宝 50 金；武将图鉴显示与主角的友谊值（可排序）；所有金币收支都会在结算弹窗（与经验同屏）或即时提示中明示金额与余额。**加深友谊**有两种途径：在武将详情页的装备栏为其选择宝物即视为赠送（无需另开弹窗，且史实武将的装备栏没有「卸下」选项、只能以新换旧），增幅按宝物稀有度 +10/+30/+60/+100，同一件宝物只在第一次赠给某位武将时计入友谊——日后无论怎么换装、换回都不会重复计，想再赚友谊需真正获得新的宝物；若目标对应槽位已有装备，旧装备会退回玩家仓库（不会丢失）。或拜访（无需消耗，每人每【游戏日】限一次——宿营过夜后可再访，友谊随机 +1~+2）。
  - **宝物系统**：五类宝物（各 20 件历史典故命名，共 100 件）——兵器(加武力)/坐骑(加统帅)/书籍(按典籍性质固定加智力或政治)/服饰(加魅力)/奇珍，各分普通/精良/稀有/传说四档稀有度（加成 +1/+3/+6/+10）；**奇珍**不止加体力，还固定绑定「暴击率」「气血回复」「护体（残血额外减伤）」「气盛（战意回复加快）」等五种效果之一，真实影响战斗结算。历练/2v2、车轮大战、百人斩、阵营大战、组队大战、国战、世界杯获胜均有概率掉落宝物或材料（一统天下、世界杯夺冠额外必得高稀有度宝物）；**战场拾获的宝物为「未鉴定」状态，不揭示具体细节，需在「宝物库」花 50 金鉴宝后才能查看真身、装备或拆解**；养成主页「宝物库」——仓库浏览/鉴宝/拆解（拆解需二次确认）、图鉴收集进度（含各类型/各奇珍效果的属性加成区间）、每日商店购买（购买即鉴定）/花金重刷、消耗材料+金币锻造（连续 12 次未出稀有以上下一次保底，锻造品同样即得即鉴定）；首页「宝物阁」则是不受进度影响的全局总览，一次性展示全部 100 件、可编辑属性/加成（限幅 ≤15）、可自建。**400 名史实武将均可持有并穿戴宝物**——不只是队友，任意武将的详情弹窗在六维数据正下方都有一排紧凑的五槽装备栏（点击即可装备/更换，取消会回到详情页而非直接关闭）；主角的装备则在角色扮演主页管理（可自由卸下）；加成会实时计入该武将出战的任何玩法（历练/车轮/百人斩/2v2/阵营/组队/国战/世界杯），角色扮演主页、武将详情页、乃至战斗界面点头像弹出的信息里，六维数值旁都会正确标注装备带来的加成；武将图鉴列表中，任何已装备宝物的武将都会显示其携带的宝物图标（全局图鉴视图除外，不显示进度数据）。
- **多种打法**：猛攻 / 普攻 / 格挡 / 谋攻 / 束缚 / 弱化 / 疗伤 / 蓄力，采用「相克 + 属性 + 随机」结算模型，**体力归零即 KO**；完整数值公式见 [BALANCE.md](BALANCE.md)。
- **美术与音效**：水墨国风界面、雷达图属性、伤害飘字、命中震屏；**纯 Web Audio 程序化生成**的五声音阶背景音乐与战斗音效（无需任何外部音频文件）。
- **PWA**：含 `manifest.json` + `sw.js`，可「添加到主屏幕」安装为 App，离线可玩。

## 🚀 运行

任意静态服务器即可（PWA 需 http/https，不能用 `file://` 直接打开）：

```bash
# 方式一：Node http-server
npx http-server -p 8099 -c-1

# 方式二：Python
python -m http.server 8099
```

浏览器打开 `http://localhost:8099`。

## 📱 安装到手机

1. 将整个文件夹部署到任意 HTTPS 静态托管（GitHub Pages / Vercel / Netlify 等）。
2. 手机浏览器打开网址 → 菜单选择「添加到主屏幕 / 安装应用」。
3. 之后可像原生 App 一样离线启动。

## ➕ 丰富武将数据

两种方式：

1. **游戏内**：进入「武将图鉴」→「新增武将」，填写属性即可，自动持久化。
2. **改源码**：编辑 `data/generals.js`，在 `CN_GENERALS` 或 `JP_GENERALS` 数组追加：

```js
{ name: "某某", title: "称号", intro: "简介…",
  ti: 90, wu: 88, tong: 80, zhi: 70, zheng: 60, mei: 75 }
```

`side` 与 `id` 会自动补全。`ti`（体力）同时作为单挑初始 HP。

> 注：游戏内一旦改过数据库会以 `localStorage` 为准；想用源码新增项，可在图鉴页点「恢复默认」刷新。
