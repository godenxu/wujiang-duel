# 中日武将大单挑 ⚔️

三国 VS 战国 —— 中日两国名将捉对厮杀的单挑对决游戏。纯前端 HTML 实现，支持 PWA 安装到手机离线运行。

## ✨ 特性

- **武将数据库**：三国 200 名、战国 200 名（共 400 名真实历史武将），每名武将含 **姓名、称号、简介** 及 **体力 / 武力 / 统帅 / 智力 / 政治 / 魅力** 六维属性。
- **数据库可维护**：图鉴页可 **新增 / 编辑 / 删除** 武将，数据自动存入浏览器 `localStorage`；支持 **导出 / 导入 JSON**、**一键恢复默认**。
- **九大玩法模块**：
  - **经典单挑**：自选双将（或随机双将），轮换出招回合制，亲自指挥每一招；统帅定先手，魅力定暴击。
  - **车轮大战**：一员主将连续挑战由弱到强的敌将，挑战连胜纪录。
  - **百人斩·爬塔**：无尽 Roguelike——守将逐层增强、带伤攀塔，每 5 层「天赐机缘」三选一强化，最佳层数存档。
  - **2v2 主副将单挑**：副将六维的 15% 并入主将，主将危急时副将舍身驰援疗伤一次。
  - **阵营大战**：规模可选每方 50 / 100 / 全部 / 随机数量，两军自动捉对厮杀直到一方全灭（可切换详情观战；从角色扮演进入时自选武将必上阵）。
  - **组队大战**：各带兵最多 10 将，带兵攻击 / 计谋（乱阵·伏兵·安抚·驰援）/ 挑唆单挑 / 征兵；兵力条按兵量比例伸展、中间端对齐屏幕中央，受创时从武将名字一侧向中间收缩。
  - **国战·攻城略地**：每局随机生成城池地图（城市数量、双方地盘、驻军皆随机；双方武将总数相同，最多各 200 员、全部从真实武将卡池直选），Gabriel 图随机连通道路；相邻攻伐/调兵、出兵行军动画，攻城战按组队规则每方最多 10 将上阵（超编随机选拔：攻方其余留守出发城，守方其余城内候命、城破时退往相邻友城），上阵败方全军覆没、阵亡不复活，占领全图一统天下。
  - **武将世界杯（32 强）**：随机分组 → 小组循环赛 → 单败淘汰；淘汰赛开打前可填**竞猜预测表**，按命中率计分、猜中冠军翻倍；对阵图折叠屏两种形态自适应（窄屏横向、宽屏上下半区纵向汇聚）。
  - **角色扮演·武将养成**：自创武将随机六维+加点，历练/车轮/百人斩/2v2/阵营/组队/国战/世界杯全部玩法均可获得经验升级。
  - **友谊与金币**：主角与 400 名武将并肩/交手皆积累友谊（陌生→相识→好友→挚友→生死之交）；各玩法胜利赚金币；挚友可花金招募入队（生死之交免费）；队友可任 2v2 副将，同阵营队友在组队大战/国战/阵营大战中必上阵；世界杯竞猜得分直接兑换金币；金币便利消费——百人斩机缘 50 金重抽、主角单挑倒地 100 金服「回魂丹」满血续战（每场一次）、国战开局 30 金重掷地图、宝物库鉴宝 50 金；武将图鉴显示与主角的友谊值（可排序）；所有金币收支都会在结算弹窗（与经验同屏）或即时提示中明示金额与余额。**加深友谊**有两种途径：在武将详情页的装备栏为其选择宝物即视为赠送（无需另开弹窗，且史实武将的装备栏没有「卸下」选项、只能以新换旧），增幅按宝物稀有度 +10/+30/+60/+100，同一件宝物只在第一次赠给某位武将时计入友谊——日后无论怎么换装、换回都不会重复计，想再赚友谊需真正获得新的宝物；若目标对应槽位已有装备，旧装备会退回玩家仓库（不会丢失）。或每日拜访（无需消耗，每人每日限一次，友谊随机 +1~+2）。
  - **宝物系统**：五类宝物（各 20 件历史典故命名，共 100 件）——兵器(加武力)/坐骑(加统帅)/书籍(随机加智力或政治)/服饰(加魅力)/奇珍，各分普通/精良/稀有/传说四档稀有度（加成 +1/+3/+6/+10）；**奇珍**不止加体力，还固定绑定「暴击率」「气血回复」「护体（残血额外减伤）」「气盛（战意回复加快）」等五种效果之一，真实影响战斗结算。历练/2v2、车轮大战、百人斩、阵营大战、组队大战、国战、世界杯获胜均有概率掉落宝物或材料（一统天下、世界杯夺冠额外必得高稀有度宝物）；**战场拾获的宝物为「未鉴定」状态，不揭示具体细节，需在「宝物库」花 50 金鉴宝后才能查看真身、装备或拆解**；主页「宝物库」——仓库浏览/鉴宝/拆解（拆解需二次确认）、图鉴收集进度（含各类型/各奇珍效果的属性加成区间）、每日商店购买（购买即鉴定）/花金重刷、消耗材料+金币锻造（连续 12 次未出稀有以上下一次保底，锻造品同样即得即鉴定）。**400 名史实武将均可持有并穿戴宝物**——不只是队友，任意武将的详情弹窗在六维数据正下方都有一排紧凑的五槽装备栏（点击即可装备/更换，取消会回到详情页而非直接关闭）；主角的装备则在角色扮演主页管理（可自由卸下）；加成会实时计入该武将出战的任何玩法（历练/车轮/百人斩/2v2/阵营/组队/国战/世界杯），角色扮演主页、武将详情页、乃至战斗界面点头像弹出的信息里，六维数值旁都会正确标注装备带来的加成；武将图鉴列表中，任何已装备宝物的武将都会显示其携带的宝物图标。
- **多种打法**：猛攻 / 普攻 / 格挡 / 谋攻 / 束缚 / 弱化 / 疗伤 / 蓄力，采用「相克 + 属性 + 随机」结算模型，**体力归零即 KO**；完整数值公式见 [BALANCE.md](BALANCE.md)。
- **美术与音效**：水墨国风界面、雷达图属性、伤害飘字、命中震屏；**纯 Web Audio 程序化生成**的五声音阶背景音乐与战斗音效（无需任何外部音频文件）。
- **PWA**：含 `manifest.json` + `sw.js`，可「添加到主屏幕」安装为 App，离线可玩。

## 🚀 运行

任意静态服务器即可（PWA 需 http/https，不能用 `file://` 直接打开）：

```bash
# 方式一：Node http-server
npx http-server -p 8099 -c-1

# 方式二：Python
python -m http.server 8099
```

浏览器打开 `http://localhost:8099`。

## 📱 安装到手机

1. 将整个文件夹部署到任意 HTTPS 静态托管（GitHub Pages / Vercel / Netlify 等）。
2. 手机浏览器打开网址 → 菜单选择「添加到主屏幕 / 安装应用」。
3. 之后可像原生 App 一样离线启动。

## 🗂 目录结构

```
├── index.html          入口
├── manifest.json       PWA 清单
├── sw.js               Service Worker（离线缓存）
├── css/style.css       样式
├── data/generals.js    武将数据库（在数组中追加对象即可扩充）
├── js/engine.js        单挑战斗引擎
├── js/audio.js         程序化音乐与音效
├── js/app.js           界面与游戏逻辑
└── icons/              应用图标
```

## ➕ 丰富武将数据

两种方式：

1. **游戏内**：进入「武将图鉴」→「新增武将」，填写属性即可，自动持久化。
2. **改源码**：编辑 `data/generals.js`，在 `CN_GENERALS` 或 `JP_GENERALS` 数组追加：

```js
{ name: "某某", title: "称号", intro: "简介…",
  ti: 90, wu: 88, tong: 80, zhi: 70, zheng: 60, mei: 75 }
```

`side` 与 `id` 会自动补全。`ti`（体力）同时作为单挑初始 HP。

> 注：游戏内一旦改过数据库会以 `localStorage` 为准；想用源码新增项，可在图鉴页点「恢复默认」刷新。
