# 《中日武将大单挑》武将扮演版 · 改造设计方案（定稿）

> 状态：**第一、二、三期均已完成并交付**。三期全部完成后，主线玩法闭环已完整（新游戏→游历→名声→悬赏/设施/奇遇→武道会→天下无双终局）。
> 本文档为大改造的总蓝图，分期实施时以此为准；实施中的数值微调需回写本文档。

## 实施记录

- **第一期·骨架**（已完成）：新主页 5 模块（新游戏/继续游戏/武将图鉴/宝物阁/小游戏）、存档架构（全局层 `wujiang_campaign_v1` 战役元数据 + 战役层沿用既有 `wujiang_rpg_v1`/`wujiang_bond_v1`/`wujiang_armory_v1`）、开局三步向导（自创/史实+少年60%本命天赋+50%成长·巅峰原值减半成长、天下格局预选、确认覆盖）、武将图鉴全局化（`showDetail(g,{global:true})` 剥离友谊/装备数据）、宝物阁全局模板层（`wujiang_armory_global_v1` 存 overrides/custom，编辑加成限幅 ≤15）、小游戏子菜单（原 8 个玩法迁移，经审计确认不触碰战役存档）。
- **第二期·地图**（已完成）：真实地理风格 SVG 地图（中原十城+战国十城+对马岛海路，`CITIES`/`ROADS` 数据表）、移动/天数/行动力系统（`Campaign.meta.map` 存 day/ap/apMax/curCity，历练与移动各耗 1 点行动力，宿营恢复满行动力并推进一天）、折叠屏双栏布局（`min-width:600px` CSS Grid，地图占左 55%、城池面板+状态条占右 45%）、城池界面通用四功能（拜访=本地武将名录点击进详情、历练、集市、铁匠铺，后两者直接跳转宝物库对应页签）、武将按城分布与渐进登场（`buildCityAssignment` 按史实/乱入两种格局分配，`CITY_HINTS` 覆盖约 180 位主要武将的势力归属+姓名哈希兜底其余武将；开局每城现身 2~3 名，之后每宿营跨 5 天批量揭示 3~5 名新武将，不按名声分层）。
  - **导航架构调整**（第二期新增，第一期文档未预见）：天下地图成为战役的新顶层主页，原「角色扮演」主页降为地图下的"角色详情"子页（通过状态条「🎭 详情」按钮进入，返回固定回地图）；「宝物库」（仓库/商店/锻造）可从地图（集市/铁匠铺直达对应页签）或角色详情页（宝物库入口）两处进入，新增 `homeBase` 机制记录战斗/宝物库结算后应返回地图还是角色详情——`showScreen(id)` 每次显示 "rpg" 或 "map" 时更新 `homeBase`，所有"返回养成"类按钮改为统一调用 `goHome()`，从而正确返回发起该动作时所在的那一层。
  - **与设计的差异**：史实分布的势力归属为风味设计而非严谨考据——约 180 位主要武将（各方主公、五虎上将/五子良将、大名/主要家臣等）有具体城池指向，其余约 220 位次要武将按姓名哈希稳定兜底分配到同阵营的城池范围内；行动力上限暂固定为 1（名声系统等第三期成长渠道尚未实现）；悬赏榜、赶路奇遇、特色设施穿插、天下大事件、唯一奇珍、武道会均按原计划留给第三期。
- **第三期·血肉**（已完成）：
  - **名声系统**：四阶阈值（无名之辈0/小有名气60/威震一方200/名满天下500），`Campaign.addFame(n)` 增加名声并在跨阶时提示；`Campaign.recalcApMax()` 按"1+名声阶梯+武道会首冠(+1)+已装备唯一奇珍(each+1)"重算行动力上限，封顶 6，上限提升时同步补给当天行动力。名声来源：任意切磋胜利(+3)、悬赏完成(+8~35)、设施挑战胜利(+8~50)、威名榜战胜高手(+25)、世界杯按名次(+12~60，夺冠额外首冠+1上限)。
  - **行动力扩展**：`spendAP()` 通用助手，历练/车轮/百人斩/阵营/组队/国战/世界杯的 RPG 入口方法统一在真正开战前扣减（2v2 在实际提交处扣减，避免取消选副将浪费）；不再局限于第二期仅有的"历练+移动"两项。
  - **悬赏榜**：每城 2~3 条悬赏（`Campaign.reset` 时经 `genBounty` 生成），讨伐令(击败指定武将，优先本地已现身者出题)与车轮令(连胜N场)两类，约 15% 为高级悬赏(奖励更丰厚，首次达成额外掉「千里靴」)；接取耗 1 行动力并直接发起对应战斗，`RPG.onBattleEnd`/`onGauntletResult` 结算时判定完成、发奖、刷新新悬赏。
  - **赶路奇遇**：移动时约 30% 触发六选一（山贼拦路按评分测算胜率给金/路遇奇珍掉神秘宝物/行脚商队集市临时八折/世外高人加点+1/路遇名将直接加友谊/客栈奇遇临时行动力+1）；触及对马岛的海路额外 20% 概率触发风暴（延误耗尽当日行动力 或 漂流到随机港口）；对马岛集市常驻八折黑市。
  - **特色设施穿插**：`CITY_FACILITY` 表为 20 城各配一项特色设施（天下擂台/车轮战武场/百人斩道场/双人比武场/军团攻城/远征/武道会预选，7 种玩法循环覆盖 20 城），城池面板按当前所在城显示专属设施按钮；duel 类设施弹出本地已现身武将挑选框，其余直接调用对应 RPG 入口。
  - **唯一奇珍**：赤兔·千里神驹(mount，百人斩首次达 15 层掉落)、千里靴(attire，首个高级悬赏达成掉落)，`Armory.makeUniqueTreasure(key)` 固定造出传说级特殊物品，额外带 `apBonus:1` 字段供 `recalcApMax` 识别；`uniqueOwned` 标记确保全局各仅能获得一次。
  - **武道会与终局**：世界杯 32 强报名需名声达「威震一方」（`fameTierIndex>=2`）方可解锁；首次夺冠永久 +1 行动力上限；`RIVAL_NAMES` 八大高手（吕布/关羽/张飞/赵云/织田信长/武田信玄/上杉谦信/本多忠胜，按姓名匹配以兼容武将库编辑）逐一击败记录于 `rivalsDefeated`；集齐八大高手 + 至少一次武道会夺冠触发一次性「天下无双」终局弹窗，之后照常继续自由游历（`ending` 标记防止重复触发）。
  - **与设计的差异**：威名榜按姓名而非固定 id 匹配（兼容武将图鉴改名/重建默认库场景）；"行脚商队/对马黑市"统一实现为集市价格 ×0.8 而非详细的差异化折价表；特色设施与"每城限定不重复"的设想不同，出于 20 城需覆盖 7 种玩法的现实，改为循环复用（每种玩法在 2~3 座城都能找到），但仍保留"每城都有一个专属去处"的体验。

  - **切磋习得**：`RPG.trySparLearn(m, heroWon, opp)` 在既有的切磋友谊结算之后追加判定——命中概率 `1% + (对手当前友谊/300)×30%`（友谊越高概率越高，封顶 31%），命中后用 `Armory.geared()` 取双方含装备/惩罚/成长的实际六维，`DIMS.reduce` 找出败方最高一项，若严格大于胜方同一项则该项 +1：主角一侧直接改 `RPG.char.alloc[dim]`（沿用 `RPG.eff()` 判断 120 封顶，与既有加点成长走同一套读取路径，无需额外改动展示逻辑）；对手一侧新增战役内 `Campaign.mapState().statGrowth[gid][dim]` 字段——与刺杀机制引入的 `statPenalty` 结构完全对称（同为 `{ti,wu,tong,zhi,zheng,mei}`），`Armory.geared()` 补一行 `if (growth) ...+= growth[k]` 即可复用同一套装备加成叠加管线，只是符号相反（`statPenalty` 减、`statGrowth` 加）。
  - **宝物加成区间化 + 售价上调**：`Armory` 新增 `BONUS_RANGE`（`normal:[1,2], fine:[3,5], rare:[6,8], legend:[9,10]`）与 `rollBonus(rarityK)`，`makeItem` 里所有此前直接取 `rar.bonus`（含 curio 的"体魄"效果，取 `curioVals('ti')` 时同源于 `RARITIES.bonus`）的默认取值路径统一改为调用 `rollBonus`——`bonusOverride`（自定义宝物/唯一奇珍的显式覆盖值）路径不受影响，仍取固定值。`shopPrice(rarityK, discount)` 由固定表 `{normal:220, fine:520, rare:1100, legend:2600}` 改为公式 `rarityDef(rarityK).bonus × 1000`（普通1000/精良3000/稀有6000/传说10000），取的是稀有度的**基准**加成值（`RARITIES[].bonus` 本身未改，仍是 1/3/6/10）而非每件实际浮动后的加成——因为货摊/商店展示价格的时间点上具体宝物实例（及其随机加成）尚未生成，只能按档位基准定价；因此实际到手的加成有可能略高于"按标价该得的量"，是有意保留的轻度惊喜感，不做逐件动态定价。锻造 `forge()`、集市 `buyShop`/`market-buy`、悬赏与冠军奖励等一切经 `makeItem`/`shopPrice` 生成物品或计价的路径均自动获得新行为，无需分别修改调用点。

## 〇、改造总目标

把游戏从"模式菜单合集"改造为**以武将扮演为主线**的游历养成游戏，现有各战斗玩法穿插其中；同时保留一个"小游戏"入口收纳原有模式的自由版。

### 已拍板的关键决策

| 决策点 | 结论 |
|---|---|
| 史实武将开局属性 | 新建游戏时由玩家选择：少年模式（60% 起步+本命天赋）或 巅峰模式（原值开局、成长减半） |
| 老存档迁移 | **不做**。更新后旧的历练/友谊/宝物数据废弃，一切从"新游戏"开始 |
| 自建/编辑宝物数值 | 限幅 **≤15**（含奇珍特效值），输入框做上限校验 |

| 武将登场节奏 | 渐进登场，**不按名声分层**——顶级高手也可能随时现身 |
| 名声阶梯与终局 | 四阶名声 + 威名榜八大高手 + 武道会 → "天下无双"，已确认 |

## 二、新游戏 · 开局三步向导

### 第 1 步 · 选角色

- **自创武将**：沿用现有角色创建（起名 + 加点），白手起家。
- **扮演史实武将**：从图鉴任选一人，随后二选一：
  - **少年模式**（推荐）：初始六维 = 默认值 60%（四舍五入），最高两项属性为"**本命天赋**"——历练成长 +50%，且成长上限可突破默认值；
  - **巅峰模式**：默认原值开局，但历练成长减半。适合想立刻横着走的玩家。

### 第 2 步 · 选天下格局

- **史实分布**：武将按真实势力落位（蜀将在成都、武田家在甲斐、岛津在萨摩……）；
- **群雄乱入**：全部武将随机洗牌分配到 20 城，每局天下大不同。

### 第 3 步 · 确认覆盖旧档

弹确认框后建立全新战役存档：角色、金币、友谊、队伍、宝物背包、地图进度全部从零开始。

### 武将渐进登场

- 开局每城只"现身"2~3 名武将，其余处于"未登场"状态；
- **每过 5 天登场一批（3~5 人）**，弹出**天下快报**："⚡ 快报：关羽现身荆州！"；
- 登场批次完全随机，**不按名声分层**——吕布、本多忠胜级别的高手也可能开局不久就现身；
- 图鉴内可查看"已登场/未登场"状态（战役内视图），寻访名将是游历动力之一。

## 五、行动力系统

- 每天拥有 **⚡行动力**，初始上限 **1 点/天**；
- **耗 1 点**的重行动：历练、切磋/擂台/道场等一切战斗、承接悬赏战斗、移动一段路（移动同时触发赶路奇遇判定）；
- **免费**的轻行动：拜访、赠宝、逛集市、锻造、看悬赏榜、查看队伍；
- 行动力用尽后出现"🏕️ 宿营至次日"按钮，进入下一天并恢复满行动力（可提前宿营）。

### 行动力上限成长（四渠道，封顶 6 点/天）

| 渠道 | 效果 | 设计意图 |

| ⭐ 名声阶梯 | 每升一阶 +1（无名1 → 小有名气2 → 威震一方3 → 名满天下4） | 主渠道：越有名，天下事越多 |

| 🍵 客栈奇遇 | 当天临时 +1 | 赶路惊喜，不破坏上限体系 |
