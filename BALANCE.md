# 战斗数值与公式说明（BALANCE）

> 本文档汇总单挑（轮换出招回合制）的全部数值与公式，便于查阅与调整。
> 变量缩写：体=体力 · 武=武力 · 统=统帅 · 智=智力 · 政=政治 · 魅=魅力；
> 「己」=出招方，「敌/守」=承受方。实现见 `js/engine.js`，评级见 `js/app.js`。

## 一、体力 / 战意

- **体力上限 = 初始体力 = `体`**，归零即 KO。
- **战意初始** = `min(100, round(55 + 政 × 0.45))`
- **每回合末回战意** = `2 + 政 / 22`
- **出招战意消耗** = `max(2, round(招式基础耗 × (1 − min(0.34, 政/300))))`；基础耗为 0 则不耗。

| 招式 | 猛攻 | 普攻 | 格挡 | 谋攻 | 束缚 | 弱化 | 疗伤 | 蓄力 |
|---|---|---|---|---|---|---|---|---|
| 基础耗 | 20 | 0 | 0 | 7 | 12 | 10 | 11 | 4* |

\* 蓄力实际是「恢复战意」，并不扣除。

## 二、先手 / 回合

- **先手**：`spd = 统 + 随机(0~20)`，高者先手；之后双方**逐回合交替出招**。
- 被束缚的一方轮到自己时**跳过该回合**（消耗一层束缚）。

## 三、物理攻击伤害（猛攻 / 普攻）

```
base       = 武                              （只看攻方武力）
mitigation = (1 − min(0.50, 守.统/380))      （守方统帅减伤）
           × (1 − min(0.12, 守.政/1000))     （守方政治微韧性）
counter    = 相克表[招式][守方姿态]
critMul    = 蓄势命中 ? 2.0 : 1.0            （蓄力「暴发」翻倍）
luck       = 随机 0.82 ~ 1.18
atkMul     = 被「弱化」时 <1，否则 1
power      = 猛攻 1.0 / 普攻 0.5

伤害 = base × 0.32 × counter × mitigation × critMul × luck × atkMul × power
```

随后叠加**暴击**（见第五节），最后 `dmg = max(1, round(dmg))`。

## 四、谋攻伤害（智力系，无视格挡）

```
成功率 = clamp(0.55 + (己.智 − 守.智)/220, 0.12, 0.94)
  失败 → 未命中（0 伤害，"被识破"）
  成功 ↓
base       = 己.智
mitigation = 1 − min(0.55, 守.智/360)         （守方智力减伤）
其余 counter / critMul / luck / power(=1) 同物理；可暴击；忽略守方格挡
```

## 五、暴击（由魅力决定）

```
暴击率 = min(0.6, 攻.魅/280) + (处于蓄势 ? 0.35 : 0)
触发 → 伤害 × 1.7
```

猛攻 / 普攻 / 谋攻均可暴击。蓄势的 `×2.0 暴发` 与暴击 `×1.7` 可叠加。

## 六、相克倍率 counter（攻方招式 × 守方「最近姿态」）

> 守方姿态 = 它上一次的攻击招式（猛攻/普攻/谋攻）；用计、格挡后姿态记为「普通」。

| 攻＼守 | 格挡 | 谋攻 | 普攻 | 猛攻 | 蓄力 |
|---|---|---|---|---|---|
| 猛攻 | 0.55 | 1.35 | 1.05 | 1.0 | 1.45 |
| 普攻 | 0.85 | 1.05 | 1.0 | 0.95 | 1.2 |
| 谋攻 | 1.5 | 0.95 | 0.95 | 0.6 | 1.25 |

（实战中守方姿态只会是 普通/猛攻/普攻/谋攻 之一。）

## 七、格挡（必定成功）

```
减伤比例 frac = clamp(0.55 + (己.统 − 敌.武)/250, 0.2, 0.92)
被卸伤害 blocked = round(原伤害 × frac)
实际受伤 = 原伤害 − blocked
己方战意 += blocked（上限 100）   ← 卸掉的伤害全部转化为战意
```

- 仅对**下一次物理受击**生效，用后消失；**谋攻无视格挡**。

## 八、计策（智力系）

**通用成功率** = `clamp(base + (己.智 − 敌.智)/220, 0.12, 0.94)`

| 计策 | 成功率 base | 效果 |
|---|---|---|
| 束缚 | 0.30 | 敌被束缚 `(己.智 − 敌.智 > 45) ? 2 : 1` 回合（暂停出招） |
| 弱化 | 0.45 | 敌攻击 `×(1 − reduce)`，`reduce = min(0.55, 0.22 + 己.智/600)`；时长 `clamp(2 + round((己.智 − 敌.智)/30), 1, 4)` 回合 |
| 疗伤 | 0.62 | 回体力 `round(己.智 × (0.26~0.46 随机))`，不超上限 |
| 蓄力 | 0.66 | 战意 `+round(6 + 己.智 × 0.12)`，并进入「蓄势」（下次出招暴发 ×2.0、暴击率 +0.35） |
| 谋攻 | 0.55 | 见第四节（攻击型） |

- 束缚、弱化为**免费计策**：发动不占行动，同回合两者各可发动一次，之后仍可正常出招。
- 疗伤、蓄力占用本回合行动。

## 九、胜负判定

- 体力 ≤ 0 → KO。
- 自动模拟（阵营大战 / 世界杯 / 快捷观战）出手上限 160；耗尽则比剩余体力，高者胜。

## 十、评级 / 总评（图鉴与界面）

- 单维评级：`SS ≥ 100 · S ≥ 95 · A ≥ 90 · B ≥ 80 · C ≥ 70 · D ≥ 60 · E < 60`
- **总评 = 六维之和**；均值 = 和 ÷ 6（总评徽章按均值取整评级）。

## 十一、友谊系统（自选武将专属）

**获得途径**（主角出战的对应玩法中自动累计，上限 999）：

| 互动方式 | 友谊值 |
|---|---|
| 历练 / 2v2 单挑击败对手 | 对手 +5 |
| 2v2 与副将并肩获胜 | 副将 +15 |
| 组队大战同队出战 | 每名队友 +3，获胜再 +3（共 +6） |
| 百人斩击败守将 | 每名被斩守将 +4 |
| 阵营大战并肩存活到终局 | 同阵营幸存同袍各 +2 |
| 世界杯同组交手 | 同组对手各 +3 |
| 国战战至终局仍在麾下 | 同袍各 +3，获胜 +6 |
| 赠礼（见金币） | +10 / +20 / +30 |

**友谊等级**：陌生 0 → 相识 30 → 好友 80 → 挚友 150 → 生死之交 250

**招募**：
- 挚友（≥150）：可花金招募，招募费 = `武将评分 × 2` 金
- 生死之交（≥250）：免费入队
- 团队上限 = `5 + ⌊主角等级 / 10⌋`
- 队友用途：2v2 任选副将；同阵营队友在组队大战 / 国战 / 阵营大战必上阵

| 类型 | 加成属性 | 举例 |
|---|---|---|
| 兵器 ⚔️ | 武力 | 青釭剑、方天画戟、村正、正宗 |
| 坐骑 🐎 | 统帅 | 赤兔马、的卢、川中岛骏马 |

**宝物库**（主页新入口，四个页签）：

- **仓库**：浏览已获得的全部宝物；未装备的可拆解为材料（普通1／精良2／稀有3／传说5）。
- **图鉴**：按五类分组展示 40 件模板的发现进度，未发现的显示「？」。
- **商店**：每日自动刷新 6 件随机货架，明码标价购买（普通40／精良90／稀有180／传说400 金）；也可花 20 金主动重刷。
- **锻造**：每类消耗对应材料 6 份 + 40 金锻造一件随机宝物；连续 12 次未出稀有以上，下一次锻造保底出稀有以上。

**装备**：

- 主角在「角色扮演」主页、队友在「武将详情」弹窗均可见五个装备槽位，点击可装备/更换/卸下。
- 每件宝物同一时间只能装备给一位武将；属性加成直接叠加进该武将的六维，实时影响其出战的所有玩法（历练/车轮/百人斩/2v2/阵营/组队/国战/世界杯）。
- 武将图鉴列表中，已入队且装备了宝物的武将会在友谊值旁显示其携带的宝物图标。
